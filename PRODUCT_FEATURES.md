# eTrackMo Product Features

## Core Finance

- eTrackMo should function like a real personal finance app.
- Users can create multiple accounts dynamically, such as Cash, GCash, Maya, Bank, or custom account types.
- Users can freely edit each account balance.
- Users should have a transaction history page showing all transactions across accounts.

## Coop

- A Coop is a user-created group for shared financial visibility.
- A user can create a Coop and becomes that Coop owner.
- Other users join a Coop through an invite link or invite code.
- Joining a Coop creates a pending request, not automatic membership.
- The Coop owner can accept or deny join requests.
- Each user can join multiple Coops.
- Coop visibility is permission-based: users choose what account data, balances, and transactions other Coop members can see.

## Initial App Pages

- Dashboard: financial overview and recent activity.
- Accounts: create, edit, and manage account balances.
- Transactions: record income, expenses, transfers, and account adjustments.
- History: view all transactions across accounts.
- Coop: create Coops, join by code/link, manage requests, and control shared visibility.
